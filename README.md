# Установка
1. Добавьте папку "mt6701" в список включаемых директорий (include);
2. Добавьте файл mt6701.c из этой папки в сборку проекта;
3. Скопируйте файлы interface в папку Src;
4. Реализуйте функции задержки (delay) и работы с SPI или I2C в interface;
5. Скопируйте init и read функции для интерфейса (I2C или SPI) в код;
6. Вызовите simple functions init функции после настройки интерфейса в коде;
7. Теперь вы можете использовать simple functions "read" для чтения углов в градусах от 0 до 360.
